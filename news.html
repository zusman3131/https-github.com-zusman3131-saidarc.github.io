Updates & Patch Notes

Latest news, patch notes, and development updates

LATEST

Flashpoint Patch 1.22.0

March 31, 2026

A major content update introducing new weapons, enemies, map conditions, and quality-of-life improvements.

🆕 New Content

FEATURE

New Map Condition: Close Scrutiny

A new ARC Operation condition that changes how ARCs patrol and respond to player activity.

ENEMY

New ARC: Vaporizer

A dangerous new enemy type with devastating area-of-effect attacks. Avoid if possible until strategies are developed.

WEAPON

Legendary: Dolabra

Energy shotgun that delivers massive close-range damage. The ultimate indoor defense weapon.

WEAPON

Rare: Canto

Medium ammo SMG with excellent handling and fire rate. Great mid-tier option for aggressive players.

GEAR

Rare Deployable: Surge Coil

New defensive deployable that creates an electrical barrier against ARCs.

PROJECT

High-Gain Antenna

New player project that enhances communication and detection capabilities.

🤖 Enemy Changes

  • Shredders - Now spawn on all maps, increasing the threat level everywhere
  • Rocketeers - Take less damage from collisions, no longer instantly destroyed when stunned
  • Fireflies - Reduced chance for multiple Fireflies to spawn together
  • Comets - Improved behavior when stunned, making them more predictable
  • ARC Targeting - Improved target selection; ARCs switch targets more freely now
  • Detection Time - Decreased time for ARC to identify very close players

🗺️ Map Updates

Dam Battlegrounds

  • Increased Sentinel spawns
  • Adjusted enemy coverage for better flow

Buried City

  • Fixed multiple stuck spots
  • More Sentinel spawns
  • Improved navigation markers

Blue Gate

  • More Sentinel spawns
  • Reduced Comets during Hurricane condition
  • Locked Gate improvements

Spaceport

  • Increased mushroom spawns
  • Launch Tower rooftop loot increased

Stella Montis

  • Fixed missing wall geometry

⚡ Major Changes

  • Locked Room Loot - Increased loot value in all locked rooms, scaled by key rarity
  • Server Assignment - Players with custom loadouts more likely to join fresh servers
  • Crafting Improvements - Quickly acquire missing materials directly from crafting view
  • HUD Option - Hide HUD option added for console (also F10 on PC)
  • Scrappy Feeding Boost - Enhanced companion feeding mechanics

👕 New Cosmetic Sets

Wasp Hunter Set Specialized gear for taking down flying ARCs
The Brigade Set Military-inspired tactical appearance
